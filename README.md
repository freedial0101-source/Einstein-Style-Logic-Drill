## 📌 現在の状況
以下の条件を選択して、問題を解き、答えを表示することができます。
・ＡはＢである
・ＡはＢのＮ個右（or左or隣）
・上２個の条件文の否定

- しかし、「Ａは右(or左or端)からＮ番目である」という条件を、上手く機能させることができていません。
- まだ、勉強し始めて2ヵ月半であり、知らないことも多いため、いったん新たな知識の習得を優先し、この機能の導入を延期します。
