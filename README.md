# Einstein-Style-Logic-Drill
『アインシュタイン式論理脳ドリル』の問題を解くプログラムです。
現在未完成。
問題文の情報を入力すると答えの表を示してくれるプログラムを目指しています。


公開URL:
https://freedial0101-source.github.io/Einstein-Style-Logic-Drill/


現在の状況:
　不完全ではありますが、入力欄とプルダウンの選択肢の作成、答えを出す基本的なアルゴリズムまでは作りました。
ただし、答えを表示するシステムはまだ作っていません。



これから追加、修正する予定のもの:


　・答えを表示する部分の作成

　・以下の条件を選択できるようにする
　　　・右（あるいは左）から何番目か
　　　・より左（あるいは右）
　　　・～ではない

　・↑の条件が機能するように判定式を修正

　・サンプルの追加（↑が機能することが分かるもの）

　・選択肢に合わせて文章を修正する機能の修正

　・以下の不適切な入力に対応、あるいは回避するプログラム
　　　・種類別の要素の数が合わない
　　　・成立する組み合わせがない、あるいは複数ある
　　　・条件の中に出てこない特徴が２つある種類がある

　・見た目の修正（最低限違和感のないもの）
　
　・プログラムの使い方の説明



やるかは未定

　・答えを出すアルゴリズムの改良:
　　　・どの配列を結合するかで確認する条件を選ぶ
　　　・結合する前から、条件で各要素が入る場所を絞る
　　　　（「『AはBの右』ならば、Aは左端以外の部分に限られる」など）
　　　・バックトラッキングで答えを求める

　・極端なパターンへの対応
　　　（すべての条件が最後の種類の特徴について書かれているなど）
